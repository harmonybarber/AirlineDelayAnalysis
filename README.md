# Airline Delay Analysis Dashboard

## Project Overview
This project analyzes airline delays, cancellations, and operational performance trends using real-world flight data. The goal is to identify patterns in airline efficiency, seasonal delays, and airport-level performance.

##  Tools Used
- Excel (data cleaning and preprocessing)
- Tableau Public (data visualization and dashboards)
- GitHub (project documentation and version control)

## Key Questions Answered
- Which airlines experience the highest delays?
- How do delays vary by month?
- What are the main causes of flight delays?
- Which airports experience the worst delays?

## Key Insights
- Mesa Airlines consistently experienced the highest average delays from January to June.
- Weather and carrier issues are major contributors to delays
- Delay patterns vary significantly by month, indicating seasonality
- Houghton County Memorial Airport (Michigan) showed the highest average delay rates from January to June.
